# Bridgeloop

Bridgeloop is a high-end, neon-themed market intelligence platform that automatically monitors competitors, parses sentiment and pricing shifts using AI, and delivers actionable alerts.

## Features
- **Cinematic Landing Page**: 3D scroll-triggered product story with GSAP animations.
- **Insights Dashboard**: Comprehensive KPI metrics, Recharts data visualizations, and competitor timelines.
- **AI Intelligence**: Sentiment analysis and pricing variance tracking.
- **Dynamic UI**: Glassmorphic design with neon-glow cursor effects and background particles.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- GSAP & Framer Motion
- Recharts
- Lucide React

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
